#### STEP 2: Editing Your `practice.md` File

Using **markdown**, make these edits to your file:

- First and last name at top of file as size two heading
- Home school under name as size three heading
- Add horizontal rule
- Add this CSS rule as a **fenced code block**
  - Make sure you specify **css** as the language for your fenced code block
  - Use a pair of *triple backticks* to start and end your fenced code block
  - **Backtick character is located above your TAB key on your keyboard**
 
```css
header    { grid-area: header; }
nav       { grid-area: nav; }
main      { grid-area: main; }
aside     { grid-area: sidebar; }
footer    { grid-area: footer; }
```

 - Add a markdown link to the **JavaScript tutorial** home page on [w3schools.com](https://www.w3schools.com/)
 - View the `JavaScript Introduction` page
 - Add an unordered list that has two (2) list items
   - List two things JavaScript can do for you in a web page
   - **Bold** the first thing JavaScript can do for you
   - *Italicize* the second thing JavaScript can do for you
